## 3. 🤔 React 与其他框架的区别

- 专注于视图层：不像 Angular 那样提供完整的框架解决方案
- JSX 语法：将 HTML 和 JavaScript 结合，提高开发效率
- 函数式编程：Hooks 的引入使函数组件功能更加强大
- 学习曲线：相对平缓，易于上手但精通需要时间
