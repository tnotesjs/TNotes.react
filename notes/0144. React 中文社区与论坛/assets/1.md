✅ 正确提问示例：标题：useState 更新后组件未重新渲染的问题

内容：【环境信息】

- React 版本：18.2.0
- 开发工具：Vite 4.0
- 浏览器：Chrome 120

【问题描述】调用 setState 后，组件没有重新渲染，控制台无报错

【复现代码】

```tsx
function Counter() {
  const [count, setCount] = useState({ value: 0 })

  const increment = () => {
    count.value += 1 // ❌ 直接修改对象
    setCount(count)
  }

  return <button onClick={increment}>{count.value}</button>
}
```

【期望行为】点击按钮后数字应该增加

【已尝试的方案】

1. 检查了 useState 的导入
2. 确认事件绑定正确
